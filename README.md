Seamlessly blend your creativity with the power of Gemini AI. This is an advanced image editor built within Google AI Studio, featuring a suite of powerful, AI-driven tools.

## Features

- **AI Imaging**: Combine and generate images with advanced AI prompts.
- **AI Fotomax**: Enhance and upscale your photos with AI.
- **AI Mockup**: Create professional product mockups instantly.
- **AI PicTalk**: Generate natural-sounding narration for your images.
- **AI Cinemax**: Turn static images into dynamic video clips.
- **Smart Editor**: A full-featured basic image editor.
- **Converter**: Convert between various image formats.

### Local Development (Advanced)

**Prerequisites:** Node.js

1.  Install dependencies: `npm install`
2.  Set the `GEMINI_API_KEY` in a `.env` file.
3.  Run the app: `npm run dev`
